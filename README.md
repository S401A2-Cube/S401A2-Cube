# S4.A.01 - Complex Application Development

This project is a partial reconstruction and optimization of the **Cube.eu** website. The goal is to develop a high-performance, secure, and accessible web application using a modern tech stack.

## About

This project is part of our second-year university curriculum. To ensure clean and uniform code throught the project, we are following guidelines and processes described [here](Guidelines_EN.md)

## Technical Stack

### Backend (API)

* **Framework:** .NET 8
* **Database:** PostgreSQL (Code First approach)
* **Security:** JWT (JSON Web Tokens) and password encryption
* **Patterns:** Repository Pattern, Dependency Injection (DI)
* **Testing:** Unit tests with Moq
* **DevOps:** CI/CD pipeline hosted on [TBD]

### Frontend (Client)

* **Framework:** Vue.js 3
* **State Management:** Pinia
* **Routing:** Vue Router
* **Testing:** Vitest for unit tests and Cypress for End-to-End testing
* **Key Features:** Component-based architecture, API integration via Axios, and responsive design

## Key Features

* **Product Management:** Browsing of the Cube bicycle catalog.
* **Secure Authentication:** User accounts protected by JWT-based security.
* **Quality Focused:** High code coverage through automated testing.
